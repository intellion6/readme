## 🌌 About Us

> **Intellion**은 _Intelligence_ 와 _Innovation_ 의 합성어로, 기술로 세상을 지능적으로 변화시키고자 하는 우리 팀의 철학을 담고 있습니다.

우리는 창의적인 아이디어와 탄탄한 기술력을 바탕으로, 더 나은 내일을 만들어가는 개발팀입니다. 각자의 전문성을 살려 협업하며, 도전적인 문제를 함께 해결해 나갑니다.
